**Meta Data : Annotation & XML**

- Meta Data :	데이터 위의 데이터
				ex) 택배 상자 위의 스티커(상품목록)
				설정 정보를 의미
				
 XML	:	eXtensible Markup Language 확장 가능한 마크업 언어
  			설정정보 용도
  			Ajax 통신의 정보 송수신 용도 ( 최근에는 JSON 으로 대체 )
  			-> 소스코드와 메타정보가 분리되는 형태
  				-> 유지보수성 향상
  		
 Annotation	:	클래스나 변수, 메서드 위에 주석형태로 기술
 				-> 소스코드와 메타데이터가 함께 기술되는 형태 
					-> 생산성 향상(한줄로, 직관적)
