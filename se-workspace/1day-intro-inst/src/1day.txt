java는 컴파일 과정과 런타임(실행) 과정을 거쳐서 수행된다. 
컴파일(compile) : 개발자가 작성한 java code를 jvm이 읽도록 
                  기계어(bytecode)로 변환하는 과정 
		  HelloWorld.java --컴파일--> HelloWorld.class 
		  ex) javac HelloWorld.java 
런타임(runtime) : .class 기계어를 java virtual machine 이 
                  운영체제에 번역하여 프로그램을 실행시키는 것
                  ex) java HelloWorld 

  HelloWorld.java --compile-- HelloWorld.class -- runtime -- 실행결과

**1일차 주요복습키워드**
jdk  : java development kit 자바개발도구   
jvm : java virtual machine 자바가상기계 
compile : 기계어 코드 생성( .class ) 
runtime : 프로그램 실행 ( .class가 실행 )
