  **7일차 주요목차**     
   - reference data 를 저장하는 배열 
     1. 배열 선언 
     Person [] pa;
     2. 배열 생성
     pa=new Person[3];
     3. 배열 요소에 할당 
     pa[0]=new Person("아이유",22);
     sysout(pa[0].getName());//아이유
     
     선언 생성 할당 동시에 
     Person[] pa={new Person("아이유",22),new Person("설현",20)};
   
    - 오버로딩 ( overloading ) 
   : 다양한 매개변수를 정의한다 
     같은 이름의 메서드(또는 생성자)가 다양한 매개변수를 
     정의함으로 사용자 편의성을 높인다.
